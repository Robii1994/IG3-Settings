The User Import system can be used to enter a large number of users/students, teachers/admins, and classes from a formatted spreadsheet (the file must be saved as a .CSV comma delimited file type).

Please note, the Username and Password fields must contain only 0-9 a-z charcters, with a 5-character minimum - each username must be unique. To get started, click Download to see an example import file, open the file, then fill-in the appropriate data for your institution and save. Click Import Now to locate your saved import file and upload the data into the system.

For the "Record Type" field, please use "admin" to import a teacher and "user" when importing student users.
